# Poool ESLint Config

> Common ESLint rules we share between projects at Poool

## Installation

```bash
yarn add @poool/eslint-config --dev
npx install-peerdeps -Y @poool/eslint-config
```

## Usage

```json
extends: ["@poool"]
```

Available subconfigs:
- `common`
- `frontend`
- `backend`

```json
extends: ["@poool/frontend"]
```
